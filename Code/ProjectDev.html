What is a Homelab?

A homelab is an environment where you learn and test different pieces of computational software and hardware. I want you to picture some components that have to deal with your internet and a couple of different technologies that you have not tried. A homelab essentially serves as that controlled environment where you combine both and build off from there.


Server Rack

This piece of equipment organizes hardware as it easily accessible to install, troubleshoot and change specific hardware. As you can imagine, cables get messy and having a dedicated space to manage them, makes it 10x easier for you.


PDU(Power Distribution Unit)

You need more electrical sockets? Buy a PDU. You need to plug your phone/laptop charger, lamp, LED light, PC power cable? Buy a PDU. The difference between the one shown below is that you can rackmount it instead of it lying on the floor.


Patch Panel

Do you like messy cables? I'd assume not which is why adding a patch panel organizes them such as ethernet cables.


Switch

If you need more ethernet ports to connect to different devices, a switch will do the job. The one I have is from Yuanley which has 4 2.5 gb ethernet ports and 2 SFP+ ports


Shelf

Now having a shelf in a rack compares to havin a regular shelf in a kitchen. You can store what you want on it in regards to simple storage or non rackmountable hardware.






Server Rails

In order to hold our server in place, we need racks to place it on top. If were were to simply screw it in, it would work, however, there would be too much pressure on the screws as the server is long in length causing an uneven weight distribution
