Mistake #1

When making patch cables, ensure the wires are flush and even before connecting them to devices like a switch or router. If the cables aren’t straight, the connection won’t work. To fix this, use a crimp and cutter tool. This tool not only ensures the cables are flush but also crimps them securely, keeping everything in place.

Mistake #2

As we can see, the rails holding the server are unstable, which can lead to issues, as the server might eventually fall. I initially used only screws, which isn’t the proper method. Unfortunately, my rack doesn’t support rack studs, so I opted for screws and a washer instead. Surprisingly, the washer helps distribute the load and stress at the fastening point while maintaining tightness. If you’re buying a rack, ensure it fits 19-inch networking equipment and supports rack studs.
